# Changelog

## 0.1.0

- Initial release with `capture`, `identify`, `groupIdentify`, and `alias`
  methods.
- Feature flag support: `getFeatureFlag`, `isFeatureEnabled`,
  `getFeatureFlagPayload`, `getFeatureFlagResult`, `getAllFlags`, and
  `getAllFlagsAndPayloads`.
- Uses `posthog-node/edge` for edge runtime compatibility.
